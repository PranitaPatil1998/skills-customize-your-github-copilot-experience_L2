# 📘 Assignment: Text Processing in Python

## 🎯 Objective

Practice working with strings and file input/output by reading, analyzing, and transforming text files.

## 📝 Tasks

### 🛠️ Read and summarize text

#### Description
Write a program that reads a text file and provides basic summary statistics about the content.

#### Requirements
Completed program should:

- Read the contents of a file named `input.txt`.
- Count and print the total number of lines, words, and characters.
- Treat words as sequences separated by whitespace.

### 🛠️ Find and replace text

#### Description
Extend the program to perform a find-and-replace operation on the file contents.

#### Requirements
Completed program should:

- Ask the user for a word or phrase to find and a replacement string.
- Replace all occurrences of the target text in the input content.
- Write the modified content to a new file named `output.txt`.
- Print a summary indicating how many replacements were made.

### 🛠️ Analyze word frequency (optional enhancement)

#### Description
Add an optional feature to count how often each word appears in the file.

#### Requirements
Completed program should:

- Build a frequency dictionary for all words (case-insensitive).
- Print the top 5 most common words and their counts.
- Ensure punctuation does not cause words like "word," and "word" to be counted separately.
